# Nov Pax brand identity (structural summary)

> Structural walkthrough of `preview/brand-identity.html` — the `FOLIO 01 / 11` 11-folio brand kit
> (distinct from the 8-plate `Design System.html` specimen). This is a brand-reference surface, not a
> production pattern to copy literally.

## Sections (11 folios)

1. **Logo** — real N+P mark, black + white v3 (`logo-black-v3.png` / `logo-white-v3.png` in source
   `assets/`). Use the black mark on light grounds, the white mark on Onyx/Dark. Wireframe corner SVGs
   live in this skill's `assets/icons/`.
2. **Wordmark** — `nov pax` lowercase lockup (company name never changed).
3. **Brand gradients** — `--grad-aurora` (Dusk→Lilac) and siblings; the signature duotone washes.
4. **Philosophy** — calm/paradox voice ("We measure success by how quickly you forget us.").
5. **Voice** — copy rules; see `rendition-voice`.
6–11. **Social / mood boards** — art-direction references (painterly, cloud, mandala, gradient maps,
   pixel paradise), kit black/white boards, logo lotus-eye, orb grid.

## Key markup classes (from the source)

- `.logo-panel` — logo presentation cell.
- `.wordmark` — the `nov pax` lockup.
- `.hero` / `.crop` / `.kick` — Onyx hero plate with halftone moon + crop marks (Edition chrome).
- `.voice` / `.voice-grid` — philosophy + voice cards.
- Head/foot chrome: `.head` / `.foot` with `.mark`, `.ver`, `.barcode-mini` (mono code strips).

## Art direction

Surrealist brutalism × heavenly vaporwave warmth; halftone moons beside breathing orbs; mega Geist
Black 900 titles on chunky duotone plates; crop marks / registration cross / barcode / marquee chrome
frame a single Cormorant italic line. References: Bráulio Amado, Pentagram NY editorial, Hassan Rahim,
Ettore Sottsass, Mubi late-2020s rebrand.
