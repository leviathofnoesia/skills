# Nov Pax Marketing Site — UI Kit

Interactive recreation of the **novpax.org** marketing website.

## Screens covered
1. **Hero** — Full-bleed dark hero, floating nav, tagline, CTA
2. **The Cost Section** — Problem framing with bold metalcore type
3. **Signal Section** — Feature showcase with bento grid
4. **Waitlist CTA** — Gradient glass CTA section with email signup
5. **Footer** — Full footer with social links, tagline

## Design Notes
- Fonts: Cormorant Garamond (display), Inter (body), Space Grotesk (metalcore), JetBrains Mono (labels)
- Primary palette: near-void black + biolum `#2DD4BF` + magenta `#E879F9` + solar `#FBB924`
- Nav: floating pill, glassmorphic, biolum border, grid texture
- Icons: Lucide (CDN)
- Animations: CSS transitions + keyframes (Framer Motion not available in static HTML)

## Usage
Open `index.html` for the interactive prototype. Click through sections using the nav or scroll.
